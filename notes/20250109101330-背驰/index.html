股票背驰是指股价和某些技术指标（如相对强弱指数RSI、MACD、KDJ等）之间的走势出现不一致的现象，通常是指股价的走势与这些指标的走势方向相反。背驰常常用来判断市场的转折点或潜在的价格反转。

判断股票背驰的方式#

  1. 股价和技术指标的背驰背驰通常分为两种类型：顶背驰和底背驰。
    • 顶背驰（Bearish Divergence）：股价创出新高，但技术指标未能创出相应的新高，表示市场可能已经过热，反转的风险增加。此时，尽管价格继续上涨，但买盘动能在减弱，市场可能出现顶部，并有下跌的风险。
    • 底背驰（Bullish Divergence）：股价创出新低，但技术指标未能创出新低，表示市场可能过于悲观，反转的可能性增大。此时，尽管价格继续下跌，但卖压在减弱，市场可能出现底部，反弹的概率较大。
  2. 判断背驰的关键指标在实际操作中，常用的几个技术指标来判断背驰现象，包括：
    • MACD（指数平滑异同移动平均线）：MACD线和信号线的背驰是判断股价背驰的重要工具。例如，当股价创出新高时，MACD未能创出新高时，就是典型的顶背驰信号；反之，当股价创出新低，而MACD未创新低时，形成底背驰信号。
    • RSI（相对强弱指数）：RSI通常用于衡量市场是否过度买入或卖出。当股价创出新高，而RSI未能同步创出新高时，形成顶背驰；股价创出新低，而RSI未能创出新低时，形成底背驰。
    • KDJ指标：KDJ指标中的K线和D线的背驰也是常用的背驰判断工具。顶背驰和底背驰的判断方式与MACD和RSI类似。
  3. 股价与技术指标的背驰判断例子：
    • 顶背驰实例：假设股价持续上涨并创出新高，但是MACD的快线（DIF）却没有创出新的高点，而是逐渐回落。这表明虽然股价创新高，但上涨的动能在减弱，这可能是市场顶部即将到来的信号。
    • 底背驰实例：假设股价持续下跌并创出新低，但RSI却没有创出新的低点，而是逐渐上升。这表示尽管股价下行，但市场的卖压已经减弱，可能意味着市场即将反弹。
  4. 背驰的警示信号：背驰并不是一种立刻就能执行交易的信号，它通常作为市场反转的警示信号。真正的反转信号可能需要结合其他技术分析工具（如支撑位、阻力位、成交量等）来确认。如果仅仅依赖背驰，可能会在一些假信号中误操作。
  5. 背驰与趋势的关系：
    • 在趋势市场中，背驰的出现并不一定意味着马上会反转。反转信号需要时间确认，背驰信号可能只是一种短期调整的表现。
    • 在震荡市场中，背驰的信号更加有效，因为市场的波动较大，技术指标的背驰可以更早地给出转折的暗示。

总结#

股票背驰是股价和技术指标之间的反向信号，通常用于预示市场的潜在反转。背驰的判断通常依赖于常用的技术指标，如MACD、RSI、KDJ等。当股价与技术指标的走势不一致时，背驰现象可能预示着市场的顶点或底点即将到来，但背驰信号需要结合其他因素进一步确认。

MACD#

MACD（移动平均收敛/发散指标，Moving Average Convergence Divergence）是由杰拉尔德·阿佩尔（Gerald Appel）在1970年代发明的一个技术分析工具，主要用于判断股价的趋势、反转点、以及趋势的强弱。它的核心思想是利用两个不同周期的移动平均线之间的关系，来揭示市场的潜在变化。

MACD的构成#

MACD指标主要由三个部分组成：

  1. DIF线（快线）：是短期指数平滑移动平均（EMA）和长期EMA之间的差值。常见的设置是：
    • 短期EMA：12日
    • 长期EMA：26日计算公式为：\[DIF = EMA_{12}(Close) - EMA_{26}(Close)\]
  2. DEA线（慢线）：是DIF线的9日指数平滑移动平均，用来进一步平滑DIF线，减少噪音。计算公式为：\[DEA = EMA_{9}(DIF)\]
  3. MACD柱状图：是DIF线与DEA线之间的差值，表示两者之间的发散或收敛情况，柱状图通常用于衡量买卖力量的变化。其计算公式为：\[MACD = DIF - DEA\]

MACD的应用和解读#

MACD的核心是分析DIF线、DEA线及其柱状图的变化，下面是常见的应用方式：

  1. 金叉与死叉

    • 金叉（Bullish Cross）：当DIF线从下向上穿越DEA线时，形成金叉，通常被视为买入信号，意味着市场趋势可能由下行转为上涨。
    • 死叉（Bearish Cross）：当DIF线从上向下穿越DEA线时，形成死叉，通常被视为卖出信号，意味着市场趋势可能由上涨转为下跌。

  2. MACD柱状图

    • 柱状图的增长：当柱状图逐渐变长且向上增长时，表示市场的多头力量在增强，可能会有上涨的趋势。
    • 柱状图的缩短或倒转：当柱状图的长度逐渐变短或反转为负值时，意味着市场的空头力量在增强，可能会有下跌的趋势。

  3. DIF与DEA的发散/收敛

    • 发散：如果DIF和DEA的差距越来越大，表示趋势可能会加速，当前趋势强烈。
    • 收敛：如果DIF和DEA的差距逐渐缩小，表示趋势的动能减弱，市场可能会进入整理或反转阶段。

  4. 背离（Divergence）

    • 顶背离（Bearish Divergence）：股价创出新高，但MACD未能同步创出新高，可能暗示上涨动能减弱，市场可能即将反转下跌。
    • 底背离（Bullish Divergence）：股价创出新低，但MACD未能同步创出新低，可能暗示下跌动能减弱，市场可能即将反弹上涨。

实际使用中的注意事项

  • 滞后性：MACD是基于移动平均线计算的，因此其反应比股价滞后，不能即时捕捉到市场的变化。对于快速变化的市场，MACD的反应较慢，可能错失一些短期的交易机会。
  • 确认信号：MACD金叉和死叉是非常常见的信号，但为了减少假信号，通常需要配合其他技术指标或市场的整体趋势进行确认，例如：支撑/阻力位、成交量等。
  • 适用于趋势市场：MACD在趋势市场中效果较好，能够有效地捕捉到趋势的反转信号；但在震荡市场中，可能会出现多次金叉和死叉的信号，导致误判。

总结#

MACD是一个相对直观、实用的技术指标，能够帮助交易者识别市场趋势、潜在的反转点以及趋势的强弱。通过分析DIF线与DEA线的交叉、柱状图的变化以及背离现象，MACD可以为交易决策提供有力的技术依据。然而，在使用MACD时，需要注意其滞后性和假信号，最好与其他分析工具结合使用，才能提高准确性。

RSI#

RSI（相对强弱指数，Relative Strength Index）是由技术分析大师J. Welles Wilder于1978年提出的一个动量指标，用来衡量股票、期货或其他资产的超买和超卖状态，进而预测市场的可能反转。RSI是基于资产的价格变化速率和幅度，通过量化买卖力量来反映市场的强弱。

RSI的计算公式#

RSI的计算公式比较简单，核心是通过“平均上涨”和“平均下跌”来进行计算：

  1. 计算平均上涨和平均下跌：假设计算周期为14天（最常见的设定），首先计算14天内每一天的涨跌幅度。
    • 如果当天收盘价高于前一天，则为“上涨幅度”；
    • 如果当天收盘价低于前一天，则为“下跌幅度”；
    • 如果当天与前一天收盘价相等，则没有变化。
  2. 计算14天的“平均上涨”和“平均下跌”：
    • 平均上涨 = 过去14天所有上涨幅度的总和 / 14
    • 平均下跌 = 过去14天所有下跌幅度的总和 / 14
  3. 计算RS（相对强度）：\[RS = \frac{\text{平均上涨}}{\text{平均下跌}}\]
  4. 计算RSI：\[RSI = 100 - \frac{100}{1 + RS}\]

这个结果将生成一个从0到100之间的指数值。

RSI的解读#

• RSI值在70以上：当RSI值高于70时，通常认为市场已经进入超买区间，价格可能被高估，市场可能出现调整或反转的风险。 • RSI值在30以下：当RSI值低于30时，通常认为市场已经进入超卖区间，价格可能被低估，市场可能出现反弹的机会。 • RSI值在50左右：表示市场没有明显的超买或超卖，处于中性状态。

RSI的常见用法#

  1. 超买超卖信号
    • 超买：当RSI值超过70时，表明市场可能过度上涨，存在回调或反转的风险。
    • 超卖：当RSI值低于30时，表明市场可能过度下跌，存在反弹的机会。
  2. RSI的背离
    • 顶背离：股价创出新高，而RSI未能创出新高，可能意味着上涨动能减弱，市场可能会反转下跌。
    • 底背离：股价创出新低，而RSI未能创出新低，表明下跌动能减弱，市场可能会反弹上涨。
  3. RSI的趋势线
    • RSI的趋势线也可以用来发现买入或卖出的信号。例如，当RSI突破其下降趋势线时，可能意味着市场的上行趋势即将开始。
  4. RSI的中性区域
    • RSI在50附近时，市场表现为中性状态，既没有强烈的买盘，也没有强烈的卖盘。这时交易者通常需要结合其他技术指标来判断市场的未来方向。

实际应用中的注意事项#

  • 滞后性：与MACD类似，RSI是一个滞后指标，不能即时反应市场的变化。因此，RSI的信号可能会在趋势发生变化后才体现出来。
  • 震荡市场的有效性：RSI特别适合用于震荡市场，因为在这种情况下，价格往往会在一定区间内波动，RSI能够比较准确地反映出超买和超卖的状态。然而，在强烈趋势行情中，RSI可能会持续处于超买或超卖区间，而未能及时发出反转信号。
  • 结合其他指标使用：RSI单独使用时可能会出现较多的假信号，因此最好与其他技术分析工具（如MACD、均线、支撑/阻力位等）结合使用，增加信号的准确性。

RSI的优化设置#

  • 周期调整：虽然默认设置是14天，但可以根据不同的市场和个股的特点进行调整。例如，对于波动较大的股票，可以缩短周期，使用7天RSI；而对于波动较小的股票，可以延长周期，使用21天RSI。
  • 超买超卖水平调整：有些交易者在极端波动的市场中会将超买和超卖的阈值调整为80和20，而不是70和30，以减少假信号。

总结#

RSI是一个简单而强大的动量指标，能够帮助交易者识别市场的超买超卖状态及潜在的反转点。通过合理地解读RSI的数值、趋势线以及与股价的背离关系，投资者可以获得有效的买卖信号。然而，RSI也有滞后性和假信号的问题，因此在实际操作中最好与其他技术指标配合使用，以提高决策的准确性。

KDJ#

KDJ（随机指标，Stochastic Oscillator）是由乔治·莱恩（George Lane）于1950年代发明的一个动量指标，广泛应用于技术分析中，主要用来衡量市场的超买超卖状态、趋势反转信号以及市场动量的变化。KDJ指标是基于股价的收盘价相对于一定时间周期内的最高价和最低价的相对位置，帮助分析价格的动量。

KDJ的构成#

KDJ指标由三条曲线组成：

  1. K线（快速线）
  2. D线（慢速线）
  3. J线（超前线）

这些线的计算过程有一定的复杂性，以下是详细的计算步骤：

  1. 计算随机值 %K

    %K是KDJ指标的核心，它反映了当前收盘价相对于过去N个周期内的最高价和最低价的相对位置。公式如下：\[%K = \frac{\text{当前收盘价} - \text{过去N天最低价}}{\text{过去N天最高价} - \text{过去N天最低价}} \times 100\]

    其中，通常N取14天。

  2. 计算慢线 %D

    %D是%K的移动平均线，通常是%K的3日简单移动平均。计算公式为：\[%D = \text{SMA}_3(%K)\]

  3. 计算超前线 J

    J线是K线和D线的结合，J线在K线和D线之间波动，反映了市场的超买超卖程度。其计算公式为：\[J = 3 \times %K - 2 \times %D\]

    由于J线对%K和%D有加权作用，因此它能更敏感地反映市场的动量变化。

KDJ的解读#

KDJ指标的主要作用是判断市场是否处于超买或超卖状态，并发出买入或卖出信号。以下是KDJ指标常见的几种解读方法：

  1. K线与D线交叉

    • 金叉：当K线（%K）从下向上穿越D线（%D）时，形成金叉，通常被视为买入信号，表示市场可能开始上涨。
    • 死叉：当K线（%K）从上向下穿越D线（%D）时，形成死叉，通常被视为卖出信号，表示市场可能开始下跌。

  2. J线的极端值

    • 超买区：当J线大于100时，表明市场处于超买状态，价格可能过高，存在回调或反转的风险。
    • 超卖区：当J线小于0时，表明市场处于超卖状态，价格可能过低，存在反弹的机会。

  3. 背离现象

    • 顶背离：当股价创出新高，而KDJ未能同步创出新高，表明上涨动能减弱，市场可能会反转下跌。
    • 底背离：当股价创出新低，而KDJ未能同步创出新低，表明下跌动能减弱，市场可能会反弹上涨。

  4. J线与K线、D线的关系

    • J线在K线和D线之间波动：J线比K线和D线波动更剧烈，因此它常常会提前反应市场的动量变化。
    • J线的极端值与转折点：J线的极端值（如大于100或小于0）通常是市场极端情绪的反映，一旦J线回归中性区域，可能是趋势反转的信号。

KDJ的应用#

  1. 超买超卖区
    • KDJ指标可以用来判断市场的超买或超卖状态。当KDJ指标的值过高或过低时，表明市场的价格可能已经过度上涨或下跌，市场存在调整的风险。
  2. 交叉信号
    • K线和D线的交叉提供了交易信号。金叉和死叉常常是买入或卖出的信号，特别是在趋势明显的市场中，交叉信号的准确性更高。
  3. 背离信号
    • KDJ与价格的背离现象可以作为市场反转的预警信号。顶背离和底背离是判断趋势反转的重要依据。
  4. J线的使用
    • J线由于对价格变化更敏感，通常能提前发出信号。当J线处于极端值时，市场可能正在发生重大变化。

实际使用中的注意事项#

  • 滞后性：尽管KDJ是一种动量指标，但它的信号仍然具有一定的滞后性，尤其是在震荡市场中，可能会频繁出现假信号。
  • 趋势确认：KDJ通常用于捕捉市场的转折点，但为了提高交易决策的准确性，最好结合趋势确认工具（如移动平均线）和其他技术指标（如MACD、RSI等）一起使用。
  • 震荡市场的有效性：KDJ在震荡市中比较有效，因为该指标能够有效地识别超买和超卖区域。然而，在强趋势市场中，KDJ可能会长时间处于超买或超卖状态，导致信号失效。

总结#

KDJ指标是一个非常实用的动量指标，能够帮助交易者识别市场的超买超卖状态、反转信号以及市场的动量变化。通过分析K线与D线的交叉、J线的极端值、以及与价格的背离现象，KDJ能够提供重要的交易信号。然而，在实际应用中，由于其滞后性和可能出现的假信号，KDJ最好与其他技术分析工具配合使用，以提高交易的准确性和成功率。

    tags: 缠论 波动造就机会 成本低于市场则风险最低，成本做到 0 或者接近 0 基本就利于不败之地，如何降低成本？利用波动，高抛低吸。波动同时代表了风险和利润。 喜好是自我的死亡陷阱，别人会利用你的喜好制造陷阱产生波动，空头陷阱造成下跌，就会带来买入的机会，相反多都陷阱造成的上涨，会带来卖出的机会。 无波动不操作。 牛市 最简单的跟盘 所有放量突破上市首日最高价的新股 放量突破年线然后缩量回调年线的老股 趋势明确才是操作时机 也可以利用中间短期波动进行降低成本。 两种卖出时机： 长时间短线推升突然出现加速上扬 第二波（第一波火爆上涨调整后）出现背驰或放巨量（有人太高价格出货） 成分股（包含在特定指数内的个股） 选股入场 选择躁动的，也就是可能会产生波动的股票，可操作的。 入场机会： 成交量压力线 新股用上市日最高价作为标准 空头陷阱 敏感股票及时退场 如果股票特别容易被扰动，说明盘子不扎实，这种情况执行严格的资金管理管理，发现不对必须马上退出，不理会后续的诱入。 建立符合自己的有效的操作程序 避免依赖路径 用眼睛看，用心去感受，不要受消息和情绪的干扰 赚钱是一辈子的事情，而不是一锤子的买卖。全仓短线就是赌博 建立自己买卖（出入）的标准，而不是随大溜 技术指标的作用是分类 均线缠绕 任何技术指标、系统，本质上都是一个评价系统，也就是告诉你在这个系统的标准下，评价对象的强弱 任何技术指标系统的应用，首要的选择标准都和应用的资金量和操作时间有关 短线看分钟均线 大资金看 5/10 日均线 分类： 短期均线高于长期均线：多头市场，在不能做空的市场里，这就是赚钱市场。 长期均线高于短期均线：空头市场，寻找买入时机 缠绕强弱： 弱：短期均线略略走平后继续按原来趋势进行下去。（几率低） 中：短期均线靠近长期均线但不跌破或升破,然后按原来趋势继续下去。（空头市场中出现几率高，反弹结束出现于多头市场） 强：短期均线跌破或升破长期均线甚至出现反复缠绕,如胶似漆。（震荡紧随其后出现，会引发行情转折） 一旦多头市场出现缠绕较强的情况，则预示着行情转折，可能会出现陷阱，紧随其后出现震荡行情，甚至空头市场。任何行情的转折一般都是由于缠绕较强引起的。 缠绕构成介入机会 新手建议看 5/10 日线（MA5/MA10） 买入点 第一个买入点：空头市场最后一次交叉后背弛构成的空头陷阱抄底进入 第二个买入或加码的位置：多头市场后第一次交叉形成的低位。 卖出点 多头市场（短期均线高于长期均线）短期均线与长期均线交叉的时候。 买应该在第二个买点，卖应该尽量在第一个卖点。
